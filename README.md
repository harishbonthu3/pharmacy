# Pharmacy Management System

A full-stack pharmacy management system built with React + Django.

## Features

- Dashboard analytics
- Medicine inventory
- Batch stock management
- Billing system
- Customers management
- Suppliers management
- Purchase tracking
- Reports (sales, profit, expiry, low stock)

## Tech Stack

Frontend:
- React
- Vite
- TailwindCSS

Backend:
- Django
- Django REST Framework

Database:
- SQLite

## Run Backend

```bash
cd backend
venv\Scripts\activate
python manage.py runserver
